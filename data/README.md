## Data

Raw files are not included. Both sources are free.

### Moving Violations
🔗 https://opendata.dc.gov/datasets/moving-violations-issued-in-the-district-of-columbia

Download CSV → save into `Violations Data/` in project root.

### Weather Data
🔗 https://www.visualcrossing.com/weather/weather-data-services

Create free account → Historical Weather Data → 
Washington DC → CSV → one file per month → 
save into `Weather Data/` in project root.

### Required folder structure
```
dc-traffic-enforcement-analytics/
├── Violations Data/
│   └── Moving_Violations.csv
└── Weather Data/
    ├── washington_dc_2024_01.csv
    └── washington_dc_2024_02.csv
```
